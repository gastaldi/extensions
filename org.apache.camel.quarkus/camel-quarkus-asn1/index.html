Documentation

Installation

To add this extension to your project, use the relevant command in your Quarkus project directory:

Quarkus CLI

quarkus ext add org.apache.camel.quarkus:camel-quarkus-asn1

Maven

./mvnw quarkus:add-extension -Dextensions="org.apache.camel.quarkus:camel-quarkus-asn1"

Gradle

./gradlew addExtension --extensions="org.apache.camel.quarkus:camel-quarkus-asn1"
